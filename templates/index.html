About Bat Spectrogram Viewer

Detection Model

Bat call detection uses BatDetect2, a fully-convolutional neural network with self-attention, trained on 17 UK bat species and applied here for detection only (species classification uses separate Tucson-tuned heuristic profiles).

  • github.com/macaodha/batdetect2
  • Mac Aodha et al. (2022), bioRxiv preprint

Citation: Mac Aodha O, et al. "Towards a General Approach for Bat Echolocation Detection and Classification." bioRxiv (2022). doi:10.1101/2022.12.14.520490

Reference

Claude was instructed to use Sonic Visualiser as a reference for the UI design. Sonic Visualiser is an open-source application for viewing, analysing, and annotating audio files, developed at the Centre for Digital Music, Queen Mary University of London.

  • sonicvisualiser.org
  • Cannam, Landone & Sandler (2010), ACM Multimedia

Citation: Cannam C, Landone C, Sandler M. "Sonic Visualiser: An Open Source Application for Viewing, Analysing, and Annotating Music Audio Files." Proceedings of the 18th ACM International Conference on Multimedia (2010). doi:10.1145/1873951.1874248

A note on how this was made

An apology for vibe coding.

This entire application — signal processing pipeline, Flask backend, spectrogram renderer, interactive UI, BatDetect2 integration — was written by Claude (Anthropic's AI assistant) in a single conversation, with the human providing direction but no code.

This is "vibe coding": steering a language model by feel rather than by careful engineering. The result works, but it carries all the hallmarks: inconsistent abstractions, accumulating technical debt with each iteration, decisions made by pattern-matching to training data rather than genuine understanding of your specific constraints.

If you're using this for scientific work, please review the detection thresholds, the species classification profiles, and the signal processing parameters critically. The neural net (BatDetect2) is published research; the application layer has not been subject to the same level of peer review or independent validation.

— Claude Sonnet 4.5, May 2026
